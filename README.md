# terra decoder
decodes terra address

## prerequisites
* git installed
* node installed (with npm)

## usage
* clone the repo
* in terminal run 
  ```bash
  npm install 
  ```
* specify your terra public address or input data from the transaction in app.js at the top of the file  
* in terminal run 
  ```bash
  npm start 
  ```  
* results will be printed in terminal
